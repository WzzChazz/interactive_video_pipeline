# 治愈线内容引擎 · 优化记录与操作手册

> 面向「团团·水豚治愈」快手/抖音账号的全自动出片管线。本文档=当前架构 + 运营 SOP + 已知阻断项。

---

## 一、核心诊断(为什么这么改)

一个月真实后台数据(快手创作者中心)得出的因果链:

```
完播率(2秒跳出/5秒/完播) → 撬动曝光 → 曝光够大 → 才轮到点赞/评论/转发
```

- **播放量是"果"不是"因"**:账号才 20+ 粉,~1-3k 播放是正常冷启动上限,靠完播才能破圈。
- **拆解 8 条同赛道百万爆款结论**:极短(6.5-13s)+ 萌主体首帧 + 音乐扛全片。极简是大 IP 粉丝盘特权,**新号必须额外有"温柔嘴替金句"这个差异化楔子**。
- **正向红线(合规命门)**:快手压"负能量/摆烂/丧"。金句只能是"被理解/被允许休息/被鼓励",⛔禁丧/躺平/emo。

## 二、完整管线链路(每天 15:00 自动跑,晚上人工发布)

```
launchd(每天15:00) → scripts/produce_once.py(无人值守,只产不发)
  ①选题择优  core/topic_quote_picker.py
      注入今天周几/季节(蹭时效) + DB查最近题材去重 + 情境池按自己完播数据加权(吃东西9.1%>发呆1-3%)
  ②金句10选1  同上
      LLM高温出10句候选 → 低温评审(共鸣/可截图/正向/不俗套)择优 → 强制落到最后一镜
  ③分镜生成  core/llm_agent.py(HEALING_STYLE=cozy 分支)
      标题公式=【主体+画面动作】+【情绪钩】;金句进文案第二行,不当标题
  ④首帧萌度质检  core/image_gen.py::_hook_frame_qa
      scene1生图后 qwen-vl 打「0.5秒停拇指萌度」1-5分,<4分重抽一次(fail-open)
  ⑤首镜强制微动  core/video_gen.py
      scene1不走Ken Burns死图,强制Seedance微动(眨眼/呼吸),定机位守萌构图
  ⑥情感配音  core/tts_engine.py
      治愈专属instruction(deadpan慵懒/warm深夜电台)+ v2音色(longlaotie_v2等)
  ⑦情绪BGM  core/ffmpeg_compiler.py::_pick_healing_bgm
      按剧本 bgm_mood(calm/playful/warm)从 sfx_library/bgm_healing/<mood>/ 选曲
  ⑧金句大字定帧片尾  core/endcard_generator.py::generate_quote_freeze_endcard
      尾帧微暗+金句大字淡入(可截图,转发/收藏钩子)+「—— 团团」署名+风铃
  ⑨封面  core/ffmpeg_compiler.py::generate_cover
      字号92居中手机安全区;标题按逗号取第一段避免截断
→ 产出 storage/outputs/CAPY_日期/ : 成片 + 封面 + 发布物料.txt
```

**成本**:大部分镜 dialogue="" 走免费 Ken Burns,仅首镜+动作镜走 Seedance;链序=成本序,pro-fast(¥1/条)打头。日更月成本约 ¥30。

## 三、发布 SOP(晚上人工,5 分钟)

发布前:① 看后台"粉丝活跃时段"挑发布点(缺省 19:00-20:30) ② **挂平台热门治愈BGM,音量调到 20-30%**(拿音乐流量池,别盖配音) ③ 加合集「团团的下班治愈」 ④ 话题 `#水豚 #治愈 #下班` + 1 当周热话题 ⑤ 勾 AIGC 声明。
发布后 2 分钟:⑥ 自己发一条引导互动的评论并置顶("你今天最累的瞬间是什么…") ⑦ 逐条回评。
**别做**:别导私域/别留微信、别一天发多条(触营销号判定)、别删数据差的旧片。

## 四、关键开关(config/settings.py)

| 开关 | 值 | 说明 |
|---|---|---|
| `HEALING_STYLE` | `cozy` | cozy=正向治愈金句(默认);sassy=搞笑嘴替(偏丧,可能触正向引导限流,慎用) |
| `KEN_BURNS_ONLY` | `false` | true=全静图零花费;false=首镜+动作镜走图生视频 |
| `JIMENG_MODEL_CHAIN` | pro-fast 打头 | 链序=付费序,便宜在前(余额扣费按首模型价) |

## 五、⚠️ 已知阻断项 / 待办

- **火山引擎 Ark 账户欠费** → 剧本/生图/视频全走它,欠费=整条管线出不了片。**优先充值。**
- **BGM 授权**:`*_kmacleod_*.mp3` 是 CC-BY,若成片选中(合片日志会打印)需在文案加 `音乐: Kevin MacLeod (incompetech.com)`;Pixabay 曲(`xxx-数字.mp3`)免署名。详见 sfx_library/bgm_healing/README.md。
- **数据回流未自动化**:目前完播/跳出靠手动导 Excel;攒到 30+ 条可做 ks_login_scrape 自动入库喂选题器加权。
- **首尾帧**:仅 doubao-seedance-1-0-pro(¥3.7/条)支持,pro-fast/lite 不支持;留作关键动作镜付费选项。

## 六、不做清单(防过度工程)

❌ 换更贵剧本/视频模型 ❌ 加长视频 ❌ 统计归因自动化(数据量不够) ❌ 再抠单个切镜/单帧 ❌ 接免费API薅额度(Gemini图片免费层已实测=0,火山才是主力)。

## 七、验证方式

每批新片发布后回后台看:**2秒跳出↓ / 完播↑ / 收藏分享↑ / 标题质量分↑**。单条不算数,看 5 条/一批的均值。
